# The Internet vs The Web

**The Internet**:
- A global network made of connected computer networks, the physical infrastructure allowing communication between computers
- ARPANET created in 1969

**The Web**:
- A specific application running on the Internet
- Combination of HTTP, HTML, and URLs
- Created in 1989 by Tim Berners-Lee
