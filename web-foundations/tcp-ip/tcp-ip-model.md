# The TCP/IP Model

#### Layer 4: Application
*"What do I want to say?"*
HTTP, HTTPS, FTP, SMTP, [DNS](./dns.md), SSH

#### Layer 3: Transport
*"How do I ensure delivery?"*
[TCP](./tcp.md) (reliable, ordered) or UDP (fast, best-effort)

#### Layer 2: Internet
*"Where does it go?"*
[IP](./ip.md) (IPv4 / IPv6): addressing and routing

#### Layer 1: Link / Network Access
*"How does it physically travel?"*
Ethernet, WiFi, fiber optics

Source: [CSE 135 - Thomas Powell](https://cse135.site/foundations-overview.html)
